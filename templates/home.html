Welcome to the Opera SD library database!

Getting Started

  1. Create your account and log in. It should keep you logged in unless you reset your browser.
  2. Update the records: If you currently have books checked out on the paper form in the library, find them on the Book List and either check them out, or claim them if they have been reported as missing.
  3. Check out books:
    • Standard check-out: If you see a book is available, look for it in the library room. If you find it, check it out from the system and enjoy!
    • Report missing books: If a book was marked as available but you weren't able to find it, you can report it as missing. Once it has been found, you'll receive a notification.
    • Request checked-out books: If you want a book someone else has checked out, you can send a request email to them and work it out like civilized people. An email gets sent every time, so please don't spam the Request button. :)
  4. Check in books:
    • Standard check-in: You can view all books you have currently checked out on the Your Books page. If your book is here, simply check it in and place it back on the library shelves.
    • Claim missing books: We all forget sometimes. If your book wasn't checked out properly in the system, find it in the Book List and see if someone reported it as missing. If so, make sure to claim it first to clear the record, then check it back in as usual. If not, you can just sneak the book back in the library and no one will know. :)
  5. Give feedback: If you find any bugs or have any suggestions, please let your friendly neighborhood librarian know. If you'd like to contribute, check out the code. Thanks!

Future Improvements

  • Shelf locations: Keep track of which shelf each book is kept on, using a location identifier in the database. This will make finding books a lot easier.
  • Book metadata: Add more book metadata like author, date acquired to the library, a link to the Amazon product page, etc..
  • New books: Should have a "new books" list or announcement page.
